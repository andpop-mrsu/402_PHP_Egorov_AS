# Task02: Веб-приложение "Арифметическая прогрессия"

Веб-версия игры из спецификации на чистом PHP с использованием SQLite.

Приложение:
- генерирует арифметическую прогрессию из 10 чисел с одним пропуском `..`;
- принимает ответ игрока;
- сохраняет в базу данных имя игрока, дату игры, результат, прогрессию с пропуском, полную прогрессию, пропущенное число и ответ игрока;
- показывает историю сыгранных игр на отдельной странице.

## Структура

- `Task02/public` — корень сайта (`index.php`, `history.php`, `styles.css`)
- `Task02/src` — серверная логика (Page Controller helpers, генерация игры, БД)
- `Task02/db` — SQLite-файл базы данных (создается автоматически)

## Запуск

1. Перейти в каталог задания:

```bash
cd Task02
```

2. Запустить встроенный PHP-сервер:

```bash
php -S localhost:3000 -t public
```

3. Открыть в браузере:

- `http://localhost:3000/` — игра
- `http://localhost:3000/history.php` — история игр

## База данных

- Тип БД: SQLite
- Файл БД: `Task02/db/progression.sqlite`
- Таблица: `games`
